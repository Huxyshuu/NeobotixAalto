Project title¶

This is the folder-based copy of the project template. Copy this folder to docs/Projects/<project-slug>/ and then replace this file with the contents of ../template.md, or keep this file as the project landing page and add separate setup/operation/experiment pages.

Recommended use

For most projects, copy Projects/template.md into this folder as index.md. For larger projects, split the same content into setup.md, operation.md, experiments.md, and troubleshooting.md.
