# Overview of the documentation

Welcome to the research documentation for the **Neobotix ROX-Diff autonomous mobile robot (AMR)** used at **Aalto University**.

This documentation is intended as a practical starting point for students, researchers, and project contributors who need to understand the robot, prepare a workstation, run basic checks, and contribute repeatable experiments.

!!! info "Scope"

    These pages are a local research documentation. They do **not** replace or duplicate the Neobotix hardware manual or other official documentations.

## What is documented here?

<div class="grid cards" markdown>

-   :material-robot-industrial: **Robot overview**

    Description and specfications of the ROX-Diff platform, as well as it's software, hardware and networking configurations.

-   :material-console-line: **Basic setup**

    Publicly available ROS 2 and simulation setup notes based on Neobotix documentation, plus **Aalto-specific** setup details.

-   :lucide-play-circle: **Operating workflows**

    Operating instructions for booting up the robot, connecting to it, teleoperating and running the ROS environment.

-   :material-shield-check: **Project documentation template**

    Instructions and a template for adding new projects into the documentation to allow people of the future to contribute and work on the robot. 

</div>
